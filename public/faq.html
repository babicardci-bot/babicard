⚡ Commandes & Livraison
Comment puis-je recevoir mon code après l'achat ? +

Dès que votre paiement est confirmé, le code de votre carte cadeau vous est envoyé automatiquement par email et par SMS. La livraison est instantanée, généralement en moins de 30 secondes.

Combien de temps faut-il pour recevoir mon code ? +

La livraison est immédiate après validation du paiement. Si vous ne recevez pas votre code dans les 5 minutes, vérifiez votre dossier spam/courriers indésirables, puis contactez notre support WhatsApp.

Puis-je suivre l'état de ma commande ? +

Oui. Connectez-vous à votre compte et allez dans Mes commandes depuis votre tableau de bord. Vous pouvez y voir l'état de chaque commande et retrouver vos codes.

J'ai perdu mon code. Puis-je le récupérer ? +

Oui. Connectez-vous à votre compte → Mes commandes → cliquez sur la commande concernée. Votre code est toujours visible dans votre historique de commandes.

🎮 Utilisation des cartes
Comment utiliser une carte iTunes / App Store ? +

Sur iPhone/iPad : Ouvrez l'App Store → appuyez sur votre photo de profil → "Utiliser un code ou une carte cadeau" → entrez le code. Sur Mac/PC : ouvrez iTunes → "Compte" → "Utiliser une carte cadeau". Attention : nos cartes sont pour l'App Store américain (région USA).

Comment utiliser une carte PSN (PlayStation) ? +

Sur PS4/PS5 : allez dans le PlayStation Store → "Utiliser un code" → entrez votre code. Sur PC : connectez-vous sur playstation.com → "Utiliser un code".

Comment utiliser une carte Xbox ? +

Sur Xbox : Paramètres → "Utiliser un code". Sur PC : Microsoft Store → votre profil → "Utiliser un code". Vous pouvez aussi aller sur microsoft.com/redeem.

Les cartes ont-elles une date d'expiration ? +

La plupart de nos cartes n'expirent pas une fois achetées. Cependant, nous recommandons de les utiliser dans les 12 mois. Vérifiez les conditions propres à chaque plateforme.

🔄 Remboursement & Problèmes
Puis-je être remboursé après un achat ? +

En raison de la nature numérique des produits (codes livrés immédiatement), les remboursements ne sont généralement pas possibles une fois le code délivré. Si vous avez reçu un code invalide ou déjà utilisé, contactez-nous immédiatement sur WhatsApp — nous traiterons votre demande sous 24h.

Mon code ne fonctionne pas, que faire ? +

Contactez notre support WhatsApp au +225 07 08 59 80 80 avec une capture d'écran du code et de l'erreur. Nous vérifions et résolvons le problème rapidement.

🏪 Devenir Vendeur
Comment vendre mes cartes sur Babicard.ci ? +

Créez un compte vendeur sur notre page d'inscription vendeur. Après validation de votre dossier par notre équipe, vous pouvez ajouter vos produits et vendre vos cartes à notre communauté.

Quelle commission prenez-vous sur les ventes ? +

Une commission est prélevée sur chaque vente. Le taux exact vous sera communiqué lors de la validation de votre compte vendeur par notre équipe.
